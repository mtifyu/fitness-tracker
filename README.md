Фитнес-трекер - Система сбора и анализа данных

Проект для лабораторной работы по созданию end-to-end системы сбора данных.

Компоненты системы
1. Генератор данных 
2. PostgreSQL - база данных 
3. Redash - платформа для визуализации и анализа данных
4. Redis - кэширование для Redash

Структура данных
Таблица fitness_metrics:
- user_id (ID пользователя: 1-5)
- heart_rate (пульс: 60-180)
- steps (шаги: 0-300)
- calories_burned (калории: 0.5-15.0)
- activity_type (тип активности: running, walking, cycling, resting)
- recorded_at (время записи)

Redash: http://localhost:5001
База: fitness_db
Пользователь: postgres
Пароль: mysecretpassword

1. Уровень интенсивности тренировок

3. Активность пользователей по типам
